# Breakaway Hero

A cross-platform HTML5 game with layered image destruction mechanics.

## How to Play

### Option 1: Play Online (GitHub Pages)
1. Go to: **https://null8void.github.io/Breakaway-Hero/**
2. If the link doesn't work, enable GitHub Pages:
   - Go to: https://github.com/Null8Void/Breakaway-Hero/settings/pages
   - Set "Source" to **Deploy from a branch**
   - Set "Branch" to **main**
   - Click Save and wait 1-2 minutes

### Option 2: Play Locally
1. Open the folder in File Explorer
2. Double-click `Play.bat` (or open `index.html` in a browser)

## Controls
- **L** - Load a character image
- **Click/Drag** - Break shards
- **R** - Reset/restore shards
- **ESC** - Open menu

## Features
- Voronoi-style polygon shards
- Physics-based falling fragments
- Fusion Creator for custom layers
- Save/Load character builds
